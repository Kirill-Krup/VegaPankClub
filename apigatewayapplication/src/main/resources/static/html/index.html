FAQ

Нужна ли предоплата?

Для онлайн-бронирования списывается небольшая предоплата, в клубе можно доплатить наличными или картой.

Есть ли скидки студентам?

Да, по будням действует -15% при предъявлении студенческого билета.

Можно ли принести свою периферию?

Да, подключим вашу клавиатуру/мышь/наушники. Для безопасности – проверка на стойке.

Есть ли корпоративные или клубные мероприятия?

Мы регулярно проводим мини-турниры, тематические ночи и ивенты для компаний. Можно арендовать зал целиком.

Какие способы оплаты принимаете?

Наличные, банковские карты, Apple Pay/Google Pay. Баланс аккаунта можно пополнять на стойке.
